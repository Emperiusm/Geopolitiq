# ghidraMCP

[Permalink: ghidraMCP](https://github.com/LaurieWired/GhidraMCP#ghidramcp)

ghidraMCP is an Model Context Protocol server for allowing LLMs to autonomously reverse engineer applications. It exposes numerous tools from core Ghidra functionality to MCP clients.

ghidraMCP\_demo.mp4

# Features

[Permalink: Features](https://github.com/LaurieWired/GhidraMCP#features)

MCP Server + Ghidra Plugin

- Decompile and analyze binaries in Ghidra
- Automatically rename methods and data
- List methods, classes, imports, and exports

# Installation

## Prerequisites

[Permalink: Prerequisites](https://github.com/LaurieWired/GhidraMCP#prerequisites)

- Install [Ghidra](https://ghidra-sre.org/)
- Python3
- MCP [SDK](https://github.com/modelcontextprotocol/python-sdk)

## Ghidra

[Permalink: Ghidra](https://github.com/LaurieWired/GhidraMCP#ghidra)

First, download the latest [release](https://github.com/LaurieWired/GhidraMCP/releases) from this repository. This contains the Ghidra plugin and Python MCP client. Then, you can directly import the plugin into Ghidra.

1. Run Ghidra
2. Select `File` -\> `Install Extensions`
3. Click the `+` button
4. Select the `GhidraMCP-1-2.zip` (or your chosen version) from the downloaded release
5. Restart Ghidra
6. Make sure the GhidraMCPPlugin is enabled in `File` -\> `Configure` -\> `Developer`
7. _Optional_: Configure the port in Ghidra with `Edit` -\> `Tool Options` -\> `GhidraMCP HTTP Server`

Video Installation Guide:

ghidra\_MCP\_Install.mp4

## MCP Clients

[Permalink: MCP Clients](https://github.com/LaurieWired/GhidraMCP#mcp-clients)

Theoretically, any MCP client should work with ghidraMCP. Three examples are given below.

## Example 1: Claude Desktop

[Permalink: Example 1: Claude Desktop](https://github.com/LaurieWired/GhidraMCP#example-1-claude-desktop)

To set up Claude Desktop as a Ghidra MCP client, go to `Claude` -\> `Settings` -\> `Developer` -\> `Edit Config` -\> `claude_desktop_config.json` and add the following:

```
{
  "mcpServers": {
    "ghidra": {
      "command": "python",
      "args": [\
        "/ABSOLUTE_PATH_TO/bridge_mcp_ghidra.py",\
        "--ghidra-server",\
        "http://127.0.0.1:8080/"\
      ]
    }
  }
}
```

Alternatively, edit this file directly:

```
/Users/YOUR_USER/Library/Application Support/Claude/claude_desktop_config.json
```

The server IP and port are configurable and should be set to point to the target Ghidra instance. If not set, both will default to localhost:8080.

## Example 2: Cline

[Permalink: Example 2: Cline](https://github.com/LaurieWired/GhidraMCP#example-2-cline)

To use GhidraMCP with [Cline](https://cline.bot/), this requires manually running the MCP server as well. First run the following command:

```
python bridge_mcp_ghidra.py --transport sse --mcp-host 127.0.0.1 --mcp-port 8081 --ghidra-server http://127.0.0.1:8080/
```

The only _required_ argument is the transport. If all other arguments are unspecified, they will default to the above. Once the MCP server is running, open up Cline and select `MCP Servers` at the top.

![Cline select](https://private-user-images.githubusercontent.com/123765654/433173041-88e1f336-4729-46ee-9b81-53271e9c0ce0.png?jwt=eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJpc3MiOiJnaXRodWIuY29tIiwiYXVkIjoicmF3LmdpdGh1YnVzZXJjb250ZW50LmNvbSIsImtleSI6ImtleTUiLCJleHAiOjE3NzM5OTM2MzksIm5iZiI6MTc3Mzk5MzMzOSwicGF0aCI6Ii8xMjM3NjU2NTQvNDMzMTczMDQxLTg4ZTFmMzM2LTQ3MjktNDZlZS05YjgxLTUzMjcxZTljMGNlMC5wbmc_WC1BbXotQWxnb3JpdGhtPUFXUzQtSE1BQy1TSEEyNTYmWC1BbXotQ3JlZGVudGlhbD1BS0lBVkNPRFlMU0E1M1BRSzRaQSUyRjIwMjYwMzIwJTJGdXMtZWFzdC0xJTJGczMlMkZhd3M0X3JlcXVlc3QmWC1BbXotRGF0ZT0yMDI2MDMyMFQwNzU1MzlaJlgtQW16LUV4cGlyZXM9MzAwJlgtQW16LVNpZ25hdHVyZT0zYjI3N2YzZTAwYzlmMjNkYmRhMWE1M2VjYWU4NmIwODM4MjQyODkzMzBkZTlhOTM0YzZkN2M4NTQwYTU2MTVmJlgtQW16LVNpZ25lZEhlYWRlcnM9aG9zdCJ9.h6F6WqtJ7zrzNz1bzggKRAevTmyHpJmh-o4R9cRq8qo)

Then select `Remote Servers` and add the following, ensuring that the url matches the MCP host and port:

1. Server Name: GhidraMCP
2. Server URL: `http://127.0.0.1:8081/sse`

## Example 3: 5ire

[Permalink: Example 3: 5ire](https://github.com/LaurieWired/GhidraMCP#example-3-5ire)

Another MCP client that supports multiple models on the backend is [5ire](https://github.com/nanbingxyz/5ire). To set up GhidraMCP, open 5ire and go to `Tools` -\> `New` and set the following configurations:

1. Tool Key: ghidra
2. Name: GhidraMCP
3. Command: `python /ABSOLUTE_PATH_TO/bridge_mcp_ghidra.py`

# Building from Source

[Permalink: Building from Source](https://github.com/LaurieWired/GhidraMCP#building-from-source)

1. Copy the following files from your Ghidra directory to this project's `lib/` directory:

- `Ghidra/Features/Base/lib/Base.jar`
- `Ghidra/Features/Decompiler/lib/Decompiler.jar`
