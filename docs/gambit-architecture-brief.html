PLATFORM VISION
Gambit is a SaaS platform SDK engine. The platform provides the knowledge graph, autonomous agents, data pipeline, SSE event bus, and bootstrap API. Signal is the first product built on it — a geopolitical intelligence dashboard. Future products (supply chain, energy markets, election monitoring) share the same engine with different dashboards. Analysts interact through a 3D globe, GPU-accelerated graph explorer, and LLM-powered analytical assistant.

DYNAMIC AGENTS (runtime)
Orchestrator creates new agents on-the-fly via Opus call. Triggered by: new data sources, emerging patterns, or analyst requests. Generates system prompt, extraction schema, graph permissions, cost budget.
ORCHESTRATOR + AGENT ARCHITECTURE
  • Routing — incoming data → correct agent(s), multi-agent for cross-domain
  • Lifecycle — start/stop/restart, health monitoring, concurrency scaling
  • Cost management — per-agent daily token budgets, tier selection
  • Quality assurance — sample outputs, flag low-confidence, track accuracy
  • Conflict resolution — contradictory operations → Opus or analyst escalation
  • Priority management — crisis detection → surge relevant queues
  • Dynamic creation — new agent specs generated at runtime by Opus
AGENT MEMORY (3 TIERS)
F
Factual — The graph itself + temporal versioning. Agents query Neo4j for world state.
P
Procedural — Per-agent lessons learned doc. Updated on corrections. Prepended to system prompt.
S
Strategic — Pattern embeddings for historical parallel detection via vector similarity.
FRONTEND VIEWS
Globe View — Deck.GL + MapLibre GL + PMTiles
Primary operational view. Nodes as points at geo coords, edges as arcs across globe surface. GlobeView → MapView (flat) on zoom. Millions of points via typed arrays + GPU instancing.
Graph Explorer — Cosmograph
Full dataset GPU-accelerated graph visualization. Force-directed layout. Focused subgraph mode from any entity. Peer view to the globe, not subordinate.
Map View — Flat 2D (MapLibre GL)
Automatic transition from globe to flat map on zoom. Same Deck.GL layers, orthographic projection. Enables high-detail regional analysis.

CONFIDENCE THRESHOLDS
> 0.9 Autonomous — act, log, QA samples
0.7–0.9 Act + flag for analyst review queue
0.5–0.7 Propose, wait for analyst approval
< 0.5 Escalate to orchestrator / Opus / analyst
Mandatory review: Merging high-edge nodes, new alliances/conflicts, risk tier changes ≥2, dynamic agent first 24h, cost overage >120%.
